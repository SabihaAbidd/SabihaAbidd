## ◈ &nbsp; ENGINEERED SYSTEM

<br/>

### 🏙️ &nbsp; CityMind — AI Urban Intelligence Simulator

A city-scale AI simulation engine that models urban infrastructure, emergency response, flood propagation, crime density, and resource allocation — all running on a shared graph architecture. Seven AI algorithms implemented from scratch; no ML libraries.

**Core algorithms →** `A* Search` `CSP + AC-3` `MRV Heuristic` `Genetic Optimization` `Kruskal's MST` `K-Means Clustering` `Decision Trees`

**Stack →** `Python` `Flask` `Graph Theory` `Live Dashboard`

[![View Repo](https://img.shields.io/badge/◈%20View%20Repository-CityMindUrbanAI-00b4d8?style=for-the-badge&logo=github&logoColor=white)](https://github.com/SabihaAbidd/CityMindUrbanAI)

### ⚡ &nbsp; Bijli Bachat — Smart Energy Analytics Platform

Full-stack electricity management platform built for Pakistani households. Tracks consumption at the appliance level, forecasts bills against live tariff structures, manages budget ceilings, and integrates solar net-metering — all through a clean, layered DAO architecture.

**Stack →** `Java` `JavaFX` `MySQL` `DAO Architecture` `OOP`

[![View Repo](https://img.shields.io/badge/◈%20View%20Repository-BijliBachat-f72585?style=for-the-badge&logo=github&logoColor=white)](https://github.com/SabihaAbidd/BijliBachat)

---

### 📈 &nbsp; CPI Network Analysis — Economic Graph Intelligence

Pakistan's Consumer Price Index modeled as a weighted graph — cities as nodes, price similarity as edges. Centrality metrics identify economic hubs; temporal analysis traces exactly how inflation propagates across regions over time.

**Techniques →** `Cosine Similarity` `Z-score Normalization` `Degree / Closeness / Betweenness / Eigenvector Centrality` `Hasse Diagrams` `Partial-Order Analysis`

**Stack →** `Python` `Network Science` `Graph Analytics` `Data Visualization`

[![View Repo](https://img.shields.io/badge/◈%20View%20Repository-CPI--Network--Analysis-7209b7?style=for-the-badge&logo=github&logoColor=white)](https://github.com/SabihaAbidd/CPI-Network-Analysis-Pakistan)

---

### 🎮 &nbsp; Super Mario — x86 Assembly Platformer

Fully playable Mario-inspired platformer written in pure x86 Assembly. Custom VGA rendering pipeline, real-time gravity and collision physics, score tracking, and multi-channel audio — zero external libraries, zero abstractions. Every instruction deliberate.

**Stack →** `x86 Assembly` `VGA Mode 13h` `Hardware Audio` `Raw Memory Management`

[![View Repo](https://img.shields.io/badge/◈%20View%20Repository-super--mario--assembly--game-00b4d8?style=for-the-badge&logo=github&logoColor=white)](https://github.com/SabihaAbidd/super-mario-assembly-game)
